## About

**taskflow** is a project management with kanban & calendar.

### Key Features

- Kanban board with drag-and-drop
- Calendar view with event management
- Team collaboration features
- Task assignments and deadlines
- Priority levels and labels
- Dark/light theme toggle
- Local storage persistence

---

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage
- Python backend

---

## Controls & Usage

| Control | Action |
|---------|--------|
| `Drag cards` | Move between columns |
| `Click card` | Edit details |
| `Double-click` | Quick add |
| `Search bar` | Filter tasks |
| `Theme toggle` | Dark/light mode |
