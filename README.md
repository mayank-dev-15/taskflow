# taskflow

[![Live Demo](https://img.shields.io/badge/🚀_Live_Demo-Visit-blue?style=for-the-badge)](https://mayank-dev-15.github.io/taskflow)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
![Language](https://img.shields.io/badge/Language-JavaScript/Python-green)

Project management app with kanban board, calendar view, team collaboration, and real-time updates.

`project-management` `kanban` `calendar` `productivity`

---

## ✨ Features

- Kanban board with drag-and-drop cards
- Calendar view with event management
- Team collaboration features
- Task assignments and deadlines
- Priority levels and labels
- Search and filter tasks
- Dark/light theme toggle
- Local storage persistence
- Responsive design

## 📦 Installation

```bash
git clone https://github.com/mayank-dev-15/taskflow.git
cd taskflow
```

```bash
cd taskflow
# Open directly in browser
open index.html
# Or serve with Python
python -m http.server 8080
```

---

## 💡 Usage

- Drag cards between columns to update status
- Click a card to edit details
- Double-click empty space to add new task
- Toggle between board and calendar views
- Use search bar to filter tasks
